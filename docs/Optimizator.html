Optimizator script¶

Simple script to iterate over several possible values for the hyperparameter of the model and perform a short training, to find the best performing values.

Optimizator.Optimizator(X, y, possible_hidden_layers_number, possible_nodes_number)[source]¶

Function to cycles through a number of possible model configurations and train for a limited time, to decide on the best configuration for the final model.

Parameters: X (ndarray): Numpy array containg the featurs for model training

y (ndarray): Numpy array of the samples’ labels for training.

possible_hidden_layers_number: Array of list of integer numbers, containing all possible numbers of hidden layers of the model.

possible_nodes_number: Array of list of integer numbers, containing all possible numbers of nodes for the hidden layers of the model.
