Analysis script¶

Script to analyse the age predictions of the model. From an excel file containing “AGE_AT_SCAN” and “PREDICTED_AGE” will calculate the MSE and RMS for the prediction, both for the total and for each data collection group.

Analysis.Analysis(path_input_file: str, analysis_directory: str = None)[source]¶

Function for the data analysis of the predicted ages. Splits the model predictions for the entire data set on smaller .xlsx files for each data group, then calculates the RMS and MSE of the prediction for each group. The result are saved in a plot, shown against the resulting RMS and MSE for the complete data set.

Parameters:

path_input_file (str): path to the .xlsx containing the input data and the model’s predictions

analysis_directory (str): path to the directory to be used for the analysis. Default uses the current directory.
