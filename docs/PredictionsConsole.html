Predictions user interface¶

Basic command line user interface. Allows to either use the default (optimized) parameters for the model or to train using user defined parameters.

PredictionsConsole.PredictionsConsole(skip: bool = False)[source]¶

Rudimentary console interface for loading a trained model.

Parameters:

skip (boolean): If true the function will skip the user input and use the default values.

Returns:

The three paths to: the saved model, the normalisation array, the data to be used for the predictions
