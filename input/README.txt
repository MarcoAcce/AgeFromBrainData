This dataset contains 419 brain morphological features (volumes, thickness, area, etc.) of brain parcels and global measures, derived for 915 male subjects of the ABIDE dataset (http://fcon_1000.projects.nitrc.org/indi/abide/).
ABIDE stands for Autism Brain Imaging Data Exchange.

For each subject, the following information are specified in the fists columns:
- subject ID
- age at scan
- sex
- Full intellectual quotient (FIQ)
- diagnostic group (DX_GROUP): 1 for ASD, -1 for TD
